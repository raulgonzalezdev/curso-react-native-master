## Módulo 1: Introducción a React Native

### ¿Qué es React Native?
1.  [Una biblioteca de JavaScript para la construcción de interfaces de usuario](/examen.html)  
2.  [Un lenguaje de programación](/examen.html) 
3.  [Un framework de JavaScript para construir aplicaciones nativas para iOS y Android](/examen.html)   
4.  [Un motor de base de datos](/examen.html)  

### ¿Cuál es una desventaja de usar React Native?
1.  [No puede reutilizar el código entre iOS y Android](/examen.html)  
2.  [Es muy difícil de aprender](/examen.html)  
3.  [Algunas funciones específicas de la plataforma pueden requerir conocimientos de Swift/Obj-C o Java/Kotlin](/examen.html)  
4.  [No tiene soporte para componentes personalizados](/examen.html)  

### ¿Cuál es la principal diferencia entre React Native CLI y Expo CLI ? 
1.  [React Native CLI no permite la creación de aplicaciones](/examen.html)  
2.  [Expo CLI no permite la personalización profunda y acceso directo al código nativo](/examen.html)  
3.  [React Native CLI es un entorno de desarrollo, mientras que Expo CLI es un lenguaje de programación](/examen.html)  
4.  [No hay ninguna diferencia, son lo mismo](/examen.html)  

### ¿Qué comando se utiliza para crear un nuevo proyecto en React Native CLI? 
1.  [npm start](/examen.html)
